HOW TO SETUP:
1) Add the 6 files from the "Import" folder, to the Java Build Path
2) In the SolarSystem class, in the Init() method, go to the location
	where Planets are created. Change the addresses of the pictures
	to the correct addresses for where they are saved on your machine


HOW PROJECT WORKS:
1) Click inside window
2) Use the W,A,S,D,Q,E,R, and F keys to move around the camera
